# 🧪 Prerna's Periodic Table Explorer

An **interactive web dashboard** built with **Streamlit + Plotly** to explore the periodic table.

## Features
- 🔍 **Search** by name, symbol, or atomic number
- 🎨 **Styled element cards** with key properties

## Tech Stack
- Python
- Streamlit
- Pandas

## Data
`Elements Data Values.csv` from [Kaggle](https://www.kaggle.com/datasets)
